Welcome & Project Goal

Welcome to Project X! Your objective is to create a dataset that tests whether an LLM can solve difficult tasks that are hard to answer but easy to verify.

  • Goal: Create prompts that fool Gemini Pro 2.5 three out of three times while still being verifiable by a human.
  • Each task should involve a challenging, fact‑seeking question with a single, indisputable answer supported by verifiable evidence.
  • Answers must remain constant over time (for example, avoid time‑sensitive questions such as "Who won the World Series last year?").
  • All facts should be searchable on the web.
  • Answers must be in a single answer, in simple form. For example: Dodgers.

Critical Rules

  • No AI tools while working.
  • No plagiarism or copying. Each prompt must be unique.
  • No step‑by‑step instruction prompts: Do not tell the model what to search in which order. The model must figure out the reasoning path on its own.
